# Layout

Application layout is dynamically determined by the `route.meta.layout` value set 
in the router configuration. If not set, `BlankLayout` is used.

Layout files resides in `@/layouts` with `^[A-Z][a-zA-Z]{2,}Layout\.vue$` naming
convention.

Please take note that using `slot` with `VueRouter` is not possible. 
See <a href="https://stackoverflow.com/a/66941080" target="_blank">Stackoverflow answer</a>.

## Example Layout Config

<Source code={`
import SidebarLayout from '@/layouts/SidebarLayout.vue'
import MainView from '@/views/MainView.vue'

export default [
  {
    path: '/',
    name: 'dashboard',
    component: MainView,
    meta: {
      // Add \`layout\` to configure the layout component 
      layout: SidebarLayout,
    },
  },
]`} />
