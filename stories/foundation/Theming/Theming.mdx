# Theming

Application main styling configuration file is `src/styles/global.css`.

To customize theme, we could use theme generator at <a href="https://ui.shadcn.com/themes" target="_blank">https://ui.shadcn.com/themes</a>.

See also,

- <a href="https://ui.shadcn.com/docs/theming#list-of-variables" target="_blank">https://ui.shadcn.com/docs/theming#list-of-variables</a>

## Animation

We are using <a href="https://github.com/Wombosvideo/tw-animate-css" target="_blank">Wombosvideo/tw-animate</a>
library to handle animations.


## Charts

The configuration file for charts styling is `src/styles/chart.css`. 

Please refer to <a href="https://unovis.dev/docs/guides/theming" target="_blank">unovis theming guide</a> 
to read on how to customize the chart styling.


## Page Loading Effect

See <a href="https://github.com/rstacruz/nprogress" target="_blank">https://github.com/rstacruz/nprogress</a>.
